======================== Системные требования ===========================

	Данное программное средство предназначено для автоматизации процесса работы учебного заведения.

Что бы у вас работала программа, нужно:

1. Открыть папку EXE;

2. Переместить содержимое папки в любое удобное место;

3. Открыть файл RPO_college.exe.

Минимальные требования к аппаратном и программному обеспечению, необходимому для корректной работы программы:
•	ОС: Windows 7 / 8.1 / 10;
•	Процессор: IBM PC with a 233Mhz 486 processor;
•	Оперативная память: 64 MB ОЗУ;
•	Видеокарта: A VESA compatible Super VGA card;
•	DirectX: Версии 9.0c;
•	Место на диске: 2 GB;
•	Звуковая карта: Sound Blaster AWE32 sound card.
•	MSSQL
